# Roman Data Platform Modernization

    **Client:** Roman Inc  
    **Industry:** Insurance  
    **Duration:** 9 months  
    **Team Size:** 5  

    ## Summary
    Roman Inc engaged our consulting team to deliver a data platform modernization solution. We built scalable data pipelines, unified governance with Unity Catalog, and deployed ML models using Databricks and MLflow for advanced analytics.

    ## Tech Stack
    AWS Glue, S3, Lambda, Redshift, MLflow

    ## Outcome
    Lowered infrastructure cost by 25% using serverless ETL pipelines.
