# Wilkerson-Smith Data Quality Framework

    **Client:** Wilkerson-Smith  
    **Industry:** Retail  
    **Duration:** 12 months  
    **Team Size:** 7  

    ## Summary
    Wilkerson-Smith engaged our consulting team to deliver a data quality framework solution. We built scalable data pipelines, unified governance with Unity Catalog, and deployed ML models using Databricks and MLflow for advanced analytics.

    Kafka, Spark Streaming, FastAPI, Docker, Kubernetes
